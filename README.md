# Orion Core 🚀

Sistema profesional de gestión de inventarios y POS con Arquitectura Limpia.

## Estructura del Proyecto

- **/api**: Servidor Node.js (Clean Architecture).
- **/web**: Cliente React + Vite.
- **/db**: Configuración y esquemas de PocketBase.

## Tecnologías

React, TypeScript, Node.js, PocketBase.
